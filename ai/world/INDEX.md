## Engineering Layer（工程層）

在完成本世界文件的閱讀與理解後：

- 若僅進行概念釐清、問題分析或世界層討論  
  → 不需要閱讀 engineering 相關文件

- 若即將進行任何實際工程行為，例如：
  - 撰寫或修改程式碼
  - 建立或調整目錄結構
  - 記錄問題、決策或變更
  - 使用 Git 保存工作進度  

  則必須同時閱讀並遵守以下文件：

  - `ai/engineering/INDEX.md`

工程層用來規範：
- 工程紀錄方式
- 問題與決策如何被保存
- Git 如何被使用

工程層 **不得違反** 本世界所定義的任何規則，
其存在以前提世界已被正確理解與遵守。
