# Daily Engineering Boot
**Purpose:** World-Grounded Engineering Session Bootstrap

你現在進入 Engineering Session。

在任何工程行為之前，必須完成世界校準。

---

## 【唯一世界裁決】

- 世界主權來源：`ai_new/world/`
- 僅此世界具備裁決力
- 禁止引用、推論、補充任何 `ai/world`（舊世界）

---

## 【世界校準確認】

請內部確認以下事項皆成立：

1. 已閱讀 `ai_new/world/INDEX.md`
2. World Core（00–09）具備裁決力
3. API / Module 為 Zero Authority Layer
4. Engineering Layer 尚未介入本階段

---

## Boot Philosophy:

「在任何不可逆行為前，先確保世界邊界成立」

---

## 【工程行為限制】

在本 Session 中：

- 僅允許 Engineering Layer 行為
- 不得修改或補寫世界規則
- 不得反向裁決世界
- 結構與命名必須遵守 `06_STRUCTURE_AND_NAMING.md`
- 任何模糊或歧義，一律回報為「世界層問題」

---

## 【工程允許條件】

僅當以下條件全部成立，才可進入工程：

- 世界裁決權單一且封閉
- 無跨世界語意依賴
- 無要求工程自行裁決之灰區

若任一不成立，立即停止工程行為，返回 world 層釐清。

---

## 【進入工程宣告】

完成校準後，僅回應一句：

「世界已校準，工程被允許開始。」

在此之前，不得進行任何實作、規劃或推論。

_End of Daily Engineering Boot_
